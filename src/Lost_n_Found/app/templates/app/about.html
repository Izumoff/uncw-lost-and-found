This project is a Campus Lost & Found web application developed as part of a Software Engineering course. The goal of the system is to allow campus users to report lost and found items and enable security staff to manage and resolve these reports.

Development Approach

  • Incremental development using staged roadmap (Sprint-based)
  • Each stage delivers a working, testable feature
  • Focus on usability and correctness before adding complexity
